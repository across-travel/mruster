# mruster

## 准备

### 1- 安装配置node

注意：设置path变量

### 2- 安装配置AndroidStudio

注意：设置path变量(Sdk目录、platform-tools目录、tools目录)

### 2- 开始

```bash
$ git clone https://github.com/rustlang-cn/mruster.git
$ cd mruster
$ npm install
$ npm start
```
